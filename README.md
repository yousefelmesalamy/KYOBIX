# KYOBIX — Engineering the Digital Bridge

A modern marketing/landing page for KYOBIX, built with React 18 (via CDN) and vanilla CSS. No build step required.

## Tech Stack

- React 18 (UMD build via unpkg)
- Babel Standalone (JSX transform in-browser)
- Vanilla CSS
- Google Fonts — Inter Tight & Geist Mono

## Project Structure

```
KYOBIX/
├── index.html
├── styles.css
└── components/
    ├── App.jsx
    ├── Nav.jsx
    ├── Hero.jsx
    ├── Services.jsx
    ├── Process.jsx
    ├── Cases.jsx
    ├── MetricsTeamManifesto.jsx
    └── ContactFooter.jsx
```

## Running Locally

```bash
python3 -m http.server 3000
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.
